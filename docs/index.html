About DeadMod

Create simple asset replacement mods directly in your browser.

Currently only supports texture mods. Support for sound mods might come later.

How to use?

  1. Download Source2Viewer to find and decompile game files (pak01_dir.vpk).
  2. Edit the extracted files to your heart's content (e.g. a texture replacement mod for a hero).
  3. Select and upload your edited files here. To fill "file path", you can do either:
    • Right-click a file in Source2Viewer and select "Copy name", then click "+ Add asset" at the top and paste into the file path field.
    • Upload Deadlock's pak01_dir.vpk on the left section, then you can search for game files and click to add them.
  4. Once you're done, download your mod as a VPK and install it like any other mod.
